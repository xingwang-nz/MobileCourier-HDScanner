1. Taking a Consignment
An "Open" consignment can be assigned with the "Take" menu item.
Home Screen -> "Consignments"(bottom left) -> Tab "Open"(top middle) -> Select Consignment(s) -> "Take" (bottom right)
2. Loading a Consignment
Assigned consignments can be "Loaded" after loading completed.
Home Screen -> "Consignments"(bottom left) -> Tab "My Cons"(top left) -> Select Consignment(s) -> "Load"(bottom left)
3. Declining a Consignment
An assigned consignment can be "Declined" so that it becomes "Open" again.
Home Screen -> "Consignments"(bottom left) -> Tab "My Cons"(top right) -> Select Consignment(s) -> "Decline" (bottom right)
4. Unloading a Consignment
A loaded consignment can be "Unloaded" if necessary.
Home Screen -> "Consignments"(bottom left) -> Tab "Loaded"(top right) -> Select consignment(s) -> "Unload"(bottom right)
5. Delivering a Consignment
Loaded consignment can be delivered after the delivery has completed.
Home Screen -> "Deliver"(bottom middle) -> tap the consignment to deliver -> select delivery type(defaults to Store) -> enter print name -> tap icon Signature to let customer sign -> "Deliver" (bottom right)
6. Rejecting a Consignment
If customer refuses to accept the delivery the consignment can be rejected. The process is similar to Delivering a consignment.
Home Screen -> "Deliver"(bottom middle) -> tap the consignment to reject -> select delivery type(defaults to Store) -> slide "Accept" to left and it becomes "Reject" -> enter print name -> tap icon Signature to let customer sign -> "Deliver"(bottom right)
7. Leaving a Card - Delivery Attempt
If customer is not available for a delivery you can record a delivery attempt.
Home Screen -> "Deliver"(bottom middle) -> tap the consignment -> Tab "Leave a Card"(top right) -> enter a Note -> "Leave a Card"(bottom right)
8. Returning a Consignment
Rejected, Loaded and Undelivered consignments can be returned to supplier.
Home Screen -> "Return"(bottom right) -> select consignment(s) -> enter Notes and Print Name -> tap icon Signature for signing -> "Return"(bottom right)
